## Opening Multiple Links

### Using Chrome Tab Groups

1. Click on a topic card
2. Select the checkboxes next to the links you want to open
3. Click **"Open Selected"** at the bottom
4. All selected links open in a new Chrome tab group with the topic's color

**Benefits of Tab Groups:**
- All related tabs are grouped together
- Color-coded by topic for easy identification
- Can be collapsed/expanded in Chrome
- Keeps your browser organized

### Select All Links

Use the **"Select All"** checkbox at the top of the modal to quickly select or deselect all links at once.

## Customization

### Accessing Settings

Click the **gear icon** in the top-right corner (hover shows "Edit").

**Settings options:**
- Change Theme
- Add Topic
- Edit Topic
- Delete Topic

### Changing Color Themes

1. Open Settings
2. Scroll to **"Choose Color Palette"**
3. Click on any palette to preview it:
   - **Ocean Blue** - Professional dark blue (default)
   - **Forest Green** - Natural and calming
   - **Rose Garden** - Elegant and warm
   - **Vibrant Pink** - Bold and energetic
   - **Minimal Grey** - Clean and modern
4. Your choice is saved automatically
5. All topic cards and links update with the new colors

### Adding New Topics

1. Open Settings (gear icon)
2. Click **"+ Add Topic"**
3. Enter a topic name
4. Select an icon from the grid
5. Click **"Add Topic"**

### Editing Topics

1. Open Settings (gear icon)
2. Click **"Edit Topic"**
3. Select the topic to edit
4. Change the name or icon
5. Click **"Save Changes"**

### Deleting Topics

1. Open Settings (gear icon)
2. Click **"Delete Topic"**
3. Select the topic to delete
4. Confirm deletion

> **Warning**: Deleting a topic removes all links in it.

### Reordering Topics

1. Click and hold any topic card
2. Drag it to the desired position
3. Release to drop
4. The new order is saved automatically

## Troubleshooting

### Extension Icon Not Visible
1. Click the puzzle piece (üß©) in Chrome's toolbar
2. Find "Pandora Box"
3. Click the pin icon to keep it visible

### Links Not Opening
- Make sure URLs start with `http://` or `https://`
- Check that the website is accessible in a regular Chrome tab

### Topics Not Saving
- Make sure you have Chrome sync enabled (optional but recommended)
- Changes save automatically to Chrome local storage
- If issues persist, try reloading the extension in `chrome://extensions/`

### Tab Groups Not Creating
- Tab groups require Chrome 89 or later
- Make sure you're on a supported Chrome version

### Theme Not Changing
- Close and reopen the popup after changing themes
- If colors don't update, reload the extension

## Data & Privacy

### Where Is My Data Stored?
- All data is stored locally in Chrome's storage
- Nothing is sent to external servers
- No tracking or analytics

### Can I Export My Topics?
Currently, topics are stored in Chrome's local storage. You can:
- Enable Chrome Sync to sync across devices
- Future versions may include import/export features

### Uninstalling
If you uninstall the extension:
- All your topics and links will be removed
- No data remains in Chrome
- You can reinstall anytime and start fresh

## Frequently Asked Questions

**Q: How many topics can I create?**
A: There's no hard limit, but the popup has a scrollbar if you have many topics.

**Q: How many links per topic?**
A: Maximum 10 links per topic to keep cards clean and manageable.

**Q: Can I use folders or sub-topics?**
A: Currently, Pandora Box supports one level of organization (topics). Each topic contains links.

**Q: Does it work offline?**
A: The extension works offline, but the websites you link to require internet access.

**Q: Can I change topic colors individually?**
A: Topics use colors from the selected palette. You can choose different palettes, and topics will automatically cycle through the palette's accent colors.

**Q: How do I backup my data?**
A: Enable Chrome Sync in Chrome settings. Your topics will sync across devices where you're signed in.

**Q: Is there a mobile version?**
A: Pandora Box is a Chrome extension for desktop. Mobile Chrome doesn't support extensions yet.
